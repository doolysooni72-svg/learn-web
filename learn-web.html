HTML › الدرس الأول
ما هو HTML؟
⏱ 10 دقائق سهل لم يكتمل

HTML اختصار لـ HyperText Markup Language وهي اللغة الأساسية لبناء صفحات الويب. كل موقع على الإنترنت مبني أساساً بـ HTML.

HTML مش لغة برمجة بالمعنى الحرفي - هي لغة توصيف (Markup Language) بنستخدمها عشان نحدد هيكل المحتوى ومعناه.

💡 نكتة مفيدة

فكر في HTML زي هيكل العظام للإنسان - بتحدد الشكل الأساسي، وبعدين CSS بتجي تلبّسه وتجمّله!

المتصفح بيفهم إيه؟

لما بتكتب كود HTML وتفتحه في المتصفح (Chrome, Firefox, etc.)، المتصفح بيقرأ الكود ده وبيحوله لصفحة يقدر المستخدم يشوفها ويتفاعل معاها.

إزاي بتعمل ملف HTML؟

  • افتح أي محرر نصوص (VS Code, Notepad++)
  • احفظ الملف بامتداد .html مثال: index.html
  • افتح الملف في المتصفح وشوف النتيجة
index.html
📖 شرح الكود

<!DOCTYPE html> — بتخبر المتصفح إن الملف ده HTML5 | <html> — العنصر الجذر الأساسي | <head> — معلومات الصفحة (مش بتظهر للمستخدم) | <body> — المحتوى الظاهر للمستخدم

سؤال: HTML هي اختصار لـ؟
HyperText Making Language
HyperText Markup Language
HyperTransfer Markup Language
تحدي اكتب أول صفحة HTML خاصة بيك
اكتب هيكل HTML كامل يحتوي على: DOCTYPE، html، head بداخلها title باسمك، وbody بداخلها h1 ترحب فيه بالزوار.
<!DOCTYPE html> <html> <head> <title>اسمك هنا</title> </head> <body> <h1>أهلاً بك!</h1> </body> </html>

العناصر والوسوم (Tags)
⏱ 12 دقيقةسهللم يكتمل

الوسم (Tag) هو أساس لغة HTML. كل وسم بيوصّف نوع المحتوى الموجود بداخله.

شكل الوسم

معظم الوسوم بيجوا في زوج: وسم فتح ووسم إغلاق.

📌 مثال

<p> هذا نص </p>
↑ فتح ↑ محتوى ↑ إغلاق (لاحظ الـ /)

وسوم شائعة يجب تعرفها

  • <h1> - <h6> — عناوين من الأكبر للأصغر
  • <p> — فقرة نصية
  • <div> — حاوية عامة (block)
  • <span> — حاوية سطرية (inline)
  • <strong> — نص غامق مهم
  • <em> — نص مائل للتأكيد
  • <br> — سطر جديد (وسم ذاتي الإغلاق)
  • <hr> — خط أفقي فاصل

السمات (Attributes)

السمات بتضيف معلومات إضافية للعنصر وبتتكتب داخل وسم الفتح:

📌 مثال السمات

<p id="intro" class="text-blue">نص</p>
<img src="photo.jpg" alt="صورتي" width="300">

<div>

اكتب كود HTML يحتوي على: عنوان h2، فقرة p بداخلها كلمة بالـ strong، وخط فاصل hr.
<h2>عنواني</h2> <p>هذا نص <strong>مهم</strong> جداً.</p> <hr>

العناوين والفقرات
⏱ 10 دقائقسهللم يكتمل

العناوين والفقرات هي اللبنات الأساسية لأي محتوى نصي على الويب.

العناوين Headings

HTML بتوفر 6 مستويات من العناوين من h1 (الأكبر) إلى h6 (الأصغر). h1 هو الأهم في الصفحة وبيُستخدم مرة واحدة فقط عادةً (مهم لـ SEO).

الفقرات Paragraphs

الوسم <p> بيُستخدم لكتابة فقرات نصية. المتصفح بيضيف تلقائياً مسافة فوق وتحت الفقرة.

⚠️ تنبيه مهم

لو كتبت سطور جديدة أو مسافات زيادة في كود HTML مش هتظهر في المتصفح. لازم تستخدم <br> للسطر الجديد.

كما تشاء بلا حدود
مرة واحدة فقط
مرتين على الأكثر
تحدياعمل صفحة محتوى مقال
اكتب صفحة HTML لمقال بسيط: عنوان h1 للمقال، عنوان h2 للقسم الأول، فقرتين p للمحتوى.
<h1>عنوان المقال</h1> <h2>المقدمة</h2> <p>هذه فقرة أولى.</p> <p>هذه فقرة ثانية.</p>

الروابط والصور
⏱ 12 دقيقةمتوسطلم يكتمل

الروابط - <a> Anchor

الوسم <a> بيُستخدم لإنشاء روابط تشعبية (hyperlinks). أهم سمة فيه هي href اللي بتحدد وجهة الرابط.

  • href="https://..." — رابط لموقع خارجي
  • href="page.html" — رابط لصفحة في نفس المشروع
  • href="#section" — رابط لقسم في نفس الصفحة
  • target="_blank" — فتح الرابط في تبويب جديد

الصور - <img>

الوسم <img> وسم ذاتي الإغلاق. أهم سماته:

  • src — مسار الصورة (ضروري)
  • alt — نص بديل لو الصورة ما اتحملتش (مهم للـ accessibility وSEO)
  • width وheight — الأبعاد

اكتب HTML فيه: صورة (ممكن تستخدم via.placeholder.com)، عنوان h3 تحتها، ورابط "اقرأ أكثر" يفتح في تبويب جديد.
<img src="https://via.placeholder.com/200x100" alt="صورة"> <h3>عنوان الكارت</h3> <a href="https://example.com" target="_blank">اقرأ أكثر</a>

القوائم Lists

نوعان رئيسيان من القوائم في HTML:

  • <ul> — Unordered List (قائمة غير مرتبة - بنقاط •)
  • <ol> — Ordered List (قائمة مرتبة - بأرقام 1,2,3)
  • <li> — عنصر القائمة (List Item) يُستخدم مع كليهما
  • <dl>, <dt>, <dd> — قائمة تعريفية

الجداول Tables

  • <table> — الجدول نفسه
  • <tr> — صف (Table Row)
  • <th> — خلية رأس الجدول (Table Header)
  • <td> — خلية بيانات (Table Data)
  • <thead>, <tbody>, <tfoot> — أقسام الجدول

⏱ 15 دقيقةمتوسطلم يكتمل

النماذج (Forms) هي أهم طريقة لجمع البيانات من المستخدمين على الويب.

عناصر النماذج الأساسية

  • <form> — الحاوية الرئيسية للنموذج
  • <input type="text"> — حقل نص
  • <input type="email"> — حقل بريد إلكتروني
  • <input type="password"> — حقل كلمة مرور
  • <input type="checkbox"> — مربع اختيار
  • <input type="radio"> — زر اختيار
  • <textarea> — منطقة نص متعددة الأسطر
  • <select> / <option> — قائمة منسدلة
  • <button> — زر
  • <label> — وصف الحقل (مرتبط بـ for=id)

مقدمة في CSS
⏱ 12 دقيقةسهللم يكتمل

CSS اختصار لـ Cascading Style Sheets - بتُستخدم لتنسيق وتجميل صفحات HTML. لو HTML هو الهيكل، فـ CSS هو الملابس والديكور!

طرق إضافة CSS

  • Inline: <p style="color:red"> - مباشرة على العنصر
  • Internal: داخل وسم <style> في الـ head
  • External: ملف .css منفصل مربوط بـ <link> (الأفضل!)

بنية CSS الأساسية

📌 Selector + Property + Value

selector {
  property: value;
  property: value;
}

المحددات (Selectors)

  • p — كل عناصر <p>
  • .class-name — كل عناصر بهذا الـ class
  • #id-name — العنصر بهذا الـ id
  • * — كل العناصر
  • h1, h2, p — عناصر متعددة

<style> h1 { color: blue; } .main { background: #eee; } #footer { font-size: 12px; } </style> <h1>عنوان</h1> <p class="main">نص رئيسي</p> <p id="footer">ذيل الصفحة</p>

الألوان والخطوط

الألوان في CSS

  • أسماء: red, blue, green, tomato, coral
  • HEX: #FF0000 (أحمر) - الأكثر شيوعاً
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5) - مع شفافية

خصائص النصوص والخطوط

  • font-family — نوع الخط
  • font-size — حجم الخط (px, em, rem, %)
  • font-weight — سماكة الخط (normal, bold, 100-900)
  • font-style — نمط الخط (normal, italic)
  • text-align — محاذاة النص (left, right, center, justify)
  • text-decoration — تزيين النص (underline, none, line-through)
  • line-height — ارتفاع السطر
  • letter-spacing — المسافة بين الحروف

Box Model - نموذج الصندوق
⏱ 15 دقيقةمتوسطلم يكتمل

الـ Box Model هو من أهم المفاهيم في CSS. كل عنصر HTML هو "صندوق" له 4 طبقات من الخارج للداخل:

  • Margin — المسافة خارج العنصر (بينه وبين العناصر الأخرى)
  • Border — الحدود حول العنصر
  • Padding — المسافة بين المحتوى والحدود
  • Content — المحتوى الفعلي

خصائص مهمة

  • width / height — العرض والارتفاع
  • padding: 10px 20px — (أعلى/أسفل) (يمين/يسار)
  • margin: auto — للتوسيط الأفقي
  • border: 2px solid #000 — سمك النوع اللون
  • border-radius — تدوير الزوايا
  • box-sizing: border-box — يحسب padding وborder ضمن الـ width

ما الفرق بين padding وmargin؟

<style> .card { width: 250px; padding: 20px; border-radius: 12px; box-shadow: 0 4px 16px rgba(0,0,0,0.12); background: white; } </style> <div class="card"> <h3>عنوان الكارد</h3> <p>محتوى الكارد هنا</p> </div>

Flexbox - التخطيط المرن
⏱ 18 دقيقةمتوسطلم يكتمل

Flexbox هو نظام تخطيط CSS قوي بيخليك تتحكم في توزيع العناصر داخل حاوية بسهولة. بديل رائع للـ float القديم!

خصائص الحاوية (Container)

  • display: flex — تفعيل Flexbox
  • flex-direction — row (افتراضي) | column | row-reverse
  • justify-content — محاذاة أفقية: flex-start | center | flex-end | space-between | space-around
  • align-items — محاذاة عمودية: flex-start | center | flex-end | stretch
  • flex-wrap — wrap | nowrap (هل يكسر السطر؟)
  • gap — المسافة بين العناصر

خصائص العنصر (Item)

  • flex: 1 — يأخذ المساحة المتاحة
  • flex-grow — نسبة التمدد
  • order — ترتيب العنصر
  • align-self — محاذاة العنصر منفرداً

أي خاصية CSS تُستخدم لتوسيط العناصر أفقياً في Flexbox؟

CSS Grid - شبكة التخطيط
⏱ 18 دقيقةمتوسطلم يكتمل

CSS Grid هو نظام تخطيط ثنائي الأبعاد (صفوف وأعمدة). يمتاز عن Flexbox في إنه بيتحكم في الاتجاهين في نفس الوقت.

خصائص أساسية

  • display: grid — تفعيل Grid
  • grid-template-columns: 1fr 1fr 1fr — 3 أعمدة متساوية
  • grid-template-columns: repeat(3, 1fr) — نفس الشيء مختصر
  • grid-template-columns: 200px auto 1fr — أعمدة بأحجام مختلفة
  • grid-template-rows — تحديد الصفوف
  • gap — المسافة بين الخلايا
  • grid-column: 1 / 3 — العنصر يمتد من عمود 1 إلى 3
  • grid-row: span 2 — يمتد على صفين
grid.html
ما الفرق الرئيسي بين Flexbox وCSS Grid؟
Grid أسرع وأكثر توافقاً
Flexbox أحادي الاتجاه، Grid ثنائي الاتجاه
تحدياعمل معرض صور بـ Grid
اعمل grid بـ 3 أعمدة متساوية وgap 10px، وبداخله 6 صور placeholders بألوان مختلفة.

Transitions & Animations
⏱ 18 دقيقةمتقدملم يكتمل

Transitions - التحولات

بتضيف انتقال ناعم عند تغيير خصائص CSS (مع :hover مثلاً)

  • transition: all 0.3s ease — كل الخصائص في 0.3 ثانية
  • transition: color 0.2s, transform 0.3s — خصائص محددة
  • خيارات الـ timing: ease, linear, ease-in, ease-out, ease-in-out

Animations - الرسوم المتحركة

بتحدد حركة كاملة بـ keyframes:

  • animation-name — اسم الـ keyframe
  • animation-duration — مدة الحركة
  • animation-iteration-count: infinite — تكرار لانهائي
  • animation-direction: alternate — ذهاباً وإياباً

مقدمة في JavaScript
⏱ 12 دقيقةسهللم يكتمل

JavaScript (JS) هي لغة البرمجة الأساسية للويب. بينما HTML بتحدد المحتوى وCSS بتنسّقه، JS بتضيف التفاعل والحركة والديناميكية.

إضافة JS للصفحة

  • Inline: <button onclick="alert('مرحبا')">
  • Internal: <script> // كود JS </script> (قبل </body>)
  • External: <script src="app.js"></script>

أوامر أساسية

  • console.log() — طباعة في الـ Console
  • alert() — رسالة منبثقة

DOM Manipulation
⏱ 20 دقيقةمتقدملم يكتمل

DOM (Document Object Model) هو تمثيل HTML كشجرة من الكائنات. JavaScript تقدر تعدل على هذه الشجرة لتغيير الصفحة ديناميكياً.

اختيار العناصر

  • document.getElementById('id') — باستخدام الـ ID
  • document.querySelector('.class') — أول عنصر
  • document.querySelectorAll('p') — كل العناصر

تعديل العناصر

  • .innerHTML = '<b>نص</b>' — تغيير المحتوى (HTML)
  • .textContent = 'نص' — تغيير النص فقط
  • .style.color = 'red' — تغيير CSS
  • .classList.add('active') — إضافة class
  • .classList.remove('active') — حذف class
  • .classList.toggle('active') — تبديل class
  • .setAttribute('src', 'img.jpg') — تغيير سمة

إنشاء وإضافة عناصر

  • document.createElement('div') — إنشاء عنصر
  • parent.appendChild(child) — إضافة عنصر
  • element.remove() — حذف عنصر

الأحداث (Events)

الأحداث هي ردود أفعال الصفحة على تفاعل المستخدم. بتستخدم addEventListener:

  • click — النقر على عنصر

  • submit — إرسال نموذج
  • keydown / keyup — ضغط مفتاح
  • mouseover / mouseout — تحريك الماوس
  • load / DOMContentLoaded — تحميل الصفحة
  • scroll — التمرير
